The Product/How It Works

PoochPing (my mom's name for it and I can't think of anything better yet) should be a native app, easily installed and used by anyone, anywhere, and will be very basic in terms of what it will do and why.

When a user chooses to take their four-legged friend for an outing and allowing access to their location, and I feel the app should only be granted access while activated by the user, PoochPing will use this similarly to how runner apps are used to track progress and accurately track their live location, and at the same time calculate the distance between them and other users. This is used in conjunction with the color collar system to determine if the user should receive a push notification to alert them of a potential trigger for their dog, whether the other user has a smaller breed which your dog should avoid, a nervous dog that you need to be considerate of, or if the dog is listed as "non-friendly" and all in the area need to practice a safe social distance.

The color collar system is a tagging system to assign specific markers to the user so others can clearly see which group their dog belongs to and allow for quick decision-making that can help prevent incidents between both dogs and owners. I have tried to avoid words and colors which may negatively label certain traits and try to be as inclusive and respectful to each group. The term "non-friendly" and playful colors which are the general colors of dog toys have been used to try to accomplish a polite tagging system.

I do understand that these colors may not appeal to all and will develop a 'dark theme' as well as a colorblind-friendly icon system to ensure all can use it and avoid any miscommunication within the app itself.

User Collar

The user's location in relation to others.
This helps the user see how close other dogs are to theirs and what personalities to expect.

Pink Collar

"Non-friendly" dogs and ask for their space to be respected. They might be a rescue dog with an unknown background or their personality just doesn't gel with others.

Purple Collar

"No-small dogs" are friendly to most, they just don't know how to play nice ith smaller breeds or puppies.

Orange Collar

"No-children" dogs do great with other dogs, they can just be too much or get nervous around young childre, or get possessive over any toys.

Blue Collar

A bit nervous at first, and may come across as a bit awkward at first, just give them time to get to know you, but get edgy around large groups.

Green Collar

Easy-going and happy to say hello and play with anyone. But these dogs may come across a bit strong at first, but understand they are just playful.

Pet(User) Profile

The user profile will be more for their canine counterpart than for them. The accounts may be linked to a person via an email address or cellphone when they signup for a PoochPing account, but the details within the user profile should be compiled of the dog's details.

The dog's name, breed, age, and color collar select as well as a short bio to help others get a better idea of their personality and a picture so when close enough others can see who the profile is assigned to. This helps to avoid others judging your dog on their looks alone.

Demo

Below is an extremely basic demo constructed in javascript and uses Leaflet, and AlertifyJs and depicts how I hope the app should function. The user icon seen as a dog paw is a draggable marker for you to interact with the map and its starting location is 52.493891, 13.446395, but you may recognize the location of the address as Lohmühlenstraße 65, 12435 Berlin, Germany. The other markers have been given set starting points and move around the map to newly assigned random locations to emulate live locations of users updating. 

 
              keys.forEach(function (key) {
                if (key !== "collarAlert" && key !== "user") {
                  icons[key]["coords"][0] =
                    icons[key]["coords"][0] + (Math.random() * 0.5 - 0.25) * 0.0009;
                  icons[key]["coords"][1] =
                    icons[key]["coords"][1] + (Math.random() * 0.5 - 0.25) * 0.0009;
                  icons[key]["marker"].setLatLng(icons[key]["coords"]).update();
                }
              });

The pink marker is to mimic a user with a pink collar tag and has been given a set radius. Should any marker enter that radius the radius will change color and a notification will appear to alert the user of another dog nearby. 

 
            alertcircle.setLatLng(icons["pinkCollar"]["coords"]);
              var isInside = false;
              keys.forEach(function (key) {
                if (key !== "pinkCollar" && key !== "collarAlert") {
                  var d = map.distance(
                    icons[key]["marker"].getLatLng(),
                    alertcircle.getLatLng()
                  );
                  if (d < alertcircle.getRadius()) {
                    isInside = true;
                  }
                  alertcircle.setStyle({
                    fillColor: isInside ? "#ce1212" : "",
                  });

                  if (isInside && canBark) {
                    alertify.warning("Careful! Another dog nearby.");
                    play();

                    setTimeout(() => {
                      canBark = true;
                    }, 5000);
                    canBark = false;
                  }
                }
            });

You can either wait for one of the other markers to enter the radius or drag the user icon closer to the pink marker until the alert is triggered
